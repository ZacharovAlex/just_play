# Audio files

The backend serves these files from:

`/audio/<filename>`

For the current game config (`humorist`) the TV needs:

`/audio/humorist-rules.mp3`

## How to add the file

1. Download or export your mp3 rules audio.
2. Copy it to this folder with the exact name:
   - `humorist-rules.mp3`

Example URL:

`http://localhost:3000/audio/humorist-rules.mp3`
